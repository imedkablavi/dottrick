## ✨ Features
- 🌍 Multi-language: Arabic (RTL), English, Turkish  
- ⚡ Fast: Generates all dotted variants using a safe, optimized algorithm  
- 🧩 Practical: Copy to clipboard and save as .txt  
- 🎨 Modern UI: Animated background, glass surfaces, docked actions, responsive  
- ♿ A11y: Semantic HTML, keyboard focus, reduced-motion friendly

## 🌍 Internationalization
- Auto-detects browser language (AR/EN/TR)  
- RTL enabled for Arabic  
- Update strings in `assets/js/app.js` under `state.strings`  

---

## 🛠 Development
- Pure HTML/CSS/JS — no build step  
- Open `index.html` in a browser or serve statically  
- Security: Input sanitized and generation capped for safety  

---

## 🚢 Deployment
- Works on GitHub Pages, Netlify, Vercel  
- For Netlify, optional `_headers` can add security headers
